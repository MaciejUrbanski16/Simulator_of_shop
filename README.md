"# Simulator of shop and library" 

Simple simulator of shop and library with internal data base.

SHOP
There are two modes:
client mode,
seller mode

Client mode allows user to go into shop or library. That depends on his choose.
Being inside shop, user has opportunity to do purchases. He can choose any available at the moment things and add them to basket.
He can also resign from concrete thing and return it from basket to shop. When he decides to end purchases he gets bill with summary.

Being inside library, user has opportunity to borrow book if it is available at the moment and deadline to give back other books is not exceeded 
He can also give back borrowed books.

Seller mode is achieved after giving the password. This mode allows to editing resources of shop and library and managing to clients of library

Every data is storaged in files with .tsv extension.
